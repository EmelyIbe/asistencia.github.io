PROYECTO: Sistema Web de Asistencia de Alumnos

DESCRIPCIÓN:
Sistema para registrar asistencia de alumnos.
Usa LocalStorage como base de datos local.

TECNOLOGÍAS USADAS:
- HTML5
- CSS3
- JavaScript
- LocalStorage

FUNCIONALIDAD:
- Se ingresa el nombre del alumno en un formulario.
- Al hacer clic en "Registrar Asistencia", se guarda y muestra en la lista.
- La información se guarda localmente.
