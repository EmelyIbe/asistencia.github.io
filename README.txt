PROYECTO: Sistema Web de Asistencia de Alumnos

DESCRIPCIÓN:
Sistema estático para registrar asistencia de alumnos. Funciona 100% en el navegador sin servidor.
Usa LocalStorage como base de datos local.

TECNOLOGÍAS USADAS:
- HTML5
- CSS3
- JavaScript (sin frameworks)
- LocalStorage (para guardar los nombres de los alumnos)

FUNCIONALIDAD:
- Se ingresa el nombre del alumno en un formulario.
- Al hacer clic en "Registrar Asistencia", se guarda y muestra en la lista.
- La información se guarda localmente (persiste si recargas la página).

INSTRUCCIONES:
1. Abre el archivo "index.html" en cualquier navegador moderno.
2. Escribe el nombre de un alumno y haz clic en "Registrar Asistencia".
3. Observa cómo se agrega a la lista.

No requiere conexión a Internet ni servidor.
